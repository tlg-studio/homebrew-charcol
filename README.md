# homebrew-charcol

Homebrew tap for the [Charcol Engine](https://github.com/tlg-studio/mettle-terminal-engine) — a terminal game engine with an AI-powered game builder.

## Install

```bash
brew install tlg-studio/charcol/charcol
```

## Upgrade

```bash
brew upgrade charcol
```

## Uninstall

```bash
brew uninstall charcol
```
